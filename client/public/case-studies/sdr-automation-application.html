Overview

SDRBase is a HubSpot-backed dashboard that centralizes SDR workflows across accounts, contacts, sequences, signals, and RevOps reporting. The system combines live CRM data, AI-assisted signal classification, and a prospecting engine to help teams identify high-potential opportunities and execute faster.

Core Value

  • Consolidates T1k account operations into a single operating workspace.
  • Reduces manual prospecting and enrichment through Apollo + HubSpot orchestration.
  • Adds AI-powered news intelligence for faster outbound relevance.
  • Improves forecast visibility with RevOps dashboards and monthly revenue modeling.

System Architecture

  • Frontend: React + Vite, wouter routing, TanStack Query, shadcn/ui component system.
  • Backend: Express API with Drizzle ORM over Postgres.
  • API contracts generated from shared OpenAPI spec into typed hooks and Zod validation.
  • Core tabs: Accounts, Contacts, Sequences, Signals, News Signals, Parking Lot, and RevOps.
  • Brand system implemented with JamBase color and typography standards for enterprise white-label consistency.

AI and Automation Layer

  • News Signals pipeline classifies RSS and pasted content into deal-relevant categories.
  • LLM-assisted enrichment powers recommendation quality and outbound prioritization.
  • Outbound creative engine generates infographic and message assets with deterministic brand-safe formatting.
  • Feedback loops capture reviewer overrides to continuously improve model behavior.

RevOps and Forecasting

  • Live HubSpot sync for T1k entities and closed-won revenue signals.
  • Consolidated MRR/ARR reporting with segment-level breakdowns.
  • Hybrid ad-sales forecast model aligned to spreadsheet-style planning logic.
  • Team-level signal, sequence, and pipeline visibility for execution accountability.

Integrations

  • HubSpot private app integration for account, contact, and revenue synchronization.
  • Apollo API integration for organization and contact prospecting workflows.
  • LLM integration for classification, strategy assistance, and outbound content generation.
  • Extensible connectors for Slack approvals, Google Sheets import, and CRM pipelines.
